Instructions: Upload a new Excel (.xls) spreadsheet for a company. Make sure:

  • File is named "Company.xls", i.e. Company is Icertis
  • Spreadsheet contains data for at least 12 months (1 full year)
  • Spreadsheet tabs are labeled: "ARR by Customer", "IS (year)", "BS (year)", and "CF (year)", i.e. year is 2020 The last year does not need to have all 12 months of data, but it should be in a separate tab.
  • Each sheet has compact columns, a.k.a. there should be no empty columns in between label columns and month columns (rows can have empty rows in between)
  • For the ARR by Customer input, the leftmost column for customers is named "Customer"
  • For the financial statement inputs, i.e. IS, BS, CF, the column used for identifying fields is named "Labels"
  • All labels in the "Labels" column are exactly the same (case-sensitive), i.e. no typos, no extra spaces, same capitalization
  • For all month column labels, they are in the format dd-Mmm-yy, i.e. dd is 31, Mmm is Jan, yy is 20
  • The dates used for each month column label are the same across all input sheets, i.e. ARR by Customer has 29-Feb-16 so IS (2016) must have 29-Feb-16, not 28-Feb-16 or any other date
